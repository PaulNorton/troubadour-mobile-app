How to install an extension:

	1. Copy your extension folder here
	2. Open worona-mobile-app/extensions.json file
	3. Edit extensions.json as in the following example:

{
	"extensions": [
		"extension1_folder_name",
		"extension2_folder_name",
		"extension3_folder_name",
		"extension4_folder_name"
	]
}
